### Description

OpenClaw for macOS gives OpenClaw a native Mac home: a lightweight menu bar launcher that installs, starts, monitors, and opens your local OpenClaw workspace without asking you to live in Terminal.

The app guides you through first setup, prepares the local runtime, lets you choose where data is stored, validates the server port, configures model access, and can install official skills so your workspace is useful from the first launch.

Once installed, OpenClaw stays quietly in the menu bar. Open the browser UI, restart or stop the server, view logs, adjust preferences, export diagnostics, or uninstall cleanly from one native interface.

Highlights:

- Native macOS menu bar experience.
- Guided install or uninstall flow.
- Local OpenClaw runtime management.
- Automatic localhost health checks.
- Provider authentication setup.
- Official skills selection during setup.
- Custom data location and port.
- Launch at login support.
- Logs, diagnostics export, and preferences.
- Light branded interface by Openzen.

OpenClaw binds to localhost and stores local app data in your macOS Application Support folder. Provider credentials are kept in the macOS Keychain when configured through the launcher. Cloud model requests are sent only to the providers you choose and configure.

OpenClaw is built for developers, researchers, automation builders, and power users who want a local AI agent workspace that feels like a Mac app instead of a command-line chore.

### Описание

OpenClaw для macOS превращает локальный OpenClaw в привычное Mac-приложение: легкий menu bar launcher устанавливает, запускает, проверяет и открывает ваш локальный OpenClaw workspace без постоянной работы через Terminal.

Приложение проводит через первичную настройку, подготавливает runtime, помогает выбрать папку данных, проверяет порт сервера, настраивает доступ к AI-провайдерам и предлагает установить официальные skills уже во время установки.

После установки OpenClaw работает из menu bar. Можно открыть браузерный интерфейс, перезапустить или остановить сервер, посмотреть логи, изменить настройки, экспортировать диагностику или полностью удалить приложение и данные через нативный интерфейс.

Возможности:

- Нативный menu bar-интерфейс macOS.
- Мастер установки или удаления.
- Управление локальным OpenClaw runtime.
- Автоматические health-checks localhost-сервера.
- Настройка авторизации AI-провайдеров.
- Выбор официальных skills при установке.
- Настраиваемая папка данных и порт.
- Запуск при входе в систему.
- Логи, диагностика и Preferences.
- Светлый брендированный интерфейс Openzen.

OpenClaw работает на localhost и хранит локальные данные приложения в Application Support macOS. Учетные данные, настроенные через launcher, сохраняются в Keychain. Запросы к облачным AI-провайдерам отправляются только тем сервисам, которые пользователь сам выбрал и настроил.

OpenClaw подходит разработчикам, исследователям, automation builders и power users, которым нужен локальный AI-agent workspace в форме нормального Mac-приложения, а не набора команд в Terminal.
